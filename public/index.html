archery-api.now.sh

Archery API

Overview

Archery APi is an api for providing articles and videos links about archery.

API URL https://archery-api.now.sh/api/
API Version 1.0
Request Method GET
Response Format JSON
Github Link Archery API

Articles

Request Archery articles. Each article comes with:

  • The title of the article,
  • A link to the article page,
  • An image to show a preview of the article,
  • The source of the article.

Request:

/archery/articles

Response:
